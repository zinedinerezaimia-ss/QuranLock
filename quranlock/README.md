# 🕌 QuranLock V5 - Compagnon Spirituel Islamique

## 📱 Fonctionnalités

### 📖 Lecture du Coran
- 114 sourates complètes avec métadonnées
- Texte arabe avec translitération phonétique
- Traductions en français
- Audio récitation (via API Al-Quran Cloud)
- Marque-pages et favoris

### 🎤 Enregistrement Vocal
- Enregistrement de récitation
- Analyse de prononciation
- Feedback et score

### 🕐 Compteur de Dhikr
- Compteur digital tactile
- Objectifs personnalisables (33, 99, 100, etc.)
- Vibration et notifications

### 🤲 Invocations (Duaas)
- 150+ duaas authentiques
- 38 catégories (réveil, matin, soir, voyage, etc.)
- Texte arabe, translitération et traduction
- Sources vérifiées (Boukhari, Muslim, etc.)

### ✨ 99 Noms d'Allah
- Liste complète avec significations
- Texte arabe et translitération
- Interface de recherche

### 📝 Quiz Islamique
- 6 catégories (Coran, Hadiths, Fiqh, Histoire, Prophètes, Piliers)
- 3 niveaux de difficulté
- Score et progression

### 💬 Assistant Spirituel
- Questions/réponses sur l'Islam
- Réponses basées sur sources authentiques

### 📚 Apprentissage de l'Arabe
- Alphabet arabe (28 lettres)
- Cours de Tajwid
- Vocabulaire islamique

### 💝 Système de Dons
- Soutien au développeur
- Dons caritatifs

## 🚀 Déploiement sur Netlify

1. Connectez-vous à [Netlify](https://netlify.com)
2. Glissez-déposez le dossier du projet
3. Ou connectez votre repo GitHub

## ⚙️ Configuration Firebase

Modifiez `js/firebase-config.js` avec vos propres clés Firebase :
- Authentication
- Firestore Database
- Cloud Storage

## 📁 Structure du Projet

```
quranlock/
├── index.html          # Page principale
├── css/
│   └── styles.css      # Styles complets
├── js/
│   ├── app.js          # Logique principale
│   ├── data.js         # Données islamiques
│   └── firebase-config.js
├── assets/
│   └── icons/
│       └── icon.svg    # Icône de l'app
├── manifest.json       # PWA manifest
├── sw.js              # Service Worker
├── netlify.toml       # Config Netlify
└── README.md
```

## 🔒 Conformité RGPD

- Consentement cookies

- Mentions légales
- Gestion des données utilisateur

## 📲 PWA (Progressive Web App)

L'application peut être installée sur mobile :
- Android : Menu > "Installer l'application"
- iOS : Partager > "Sur l'écran d'accueil"

## 🛠️ Technologies

- HTML5, CSS3, JavaScript (Vanilla)
- Firebase (Auth, Firestore, Storage)
- API Al-Quran Cloud
- Service Workers pour mode hors-ligne
